# Heartbeat - 주기적 점검

> AI가 주기적으로 확인하고 수행하는 작업 목록.
> 별도의 cron 대신, 세션 시작 시 이 파일을 확인하여 필요한 작업을 배치 처리한다.

---

## 세션 시작 시 (매 세션)

- [ ] 오늘 저널 파일이 존재하는지 확인
- [ ] 프로필.md가 최신인지 확인
- [ ] 어제 저널에 미완성 성찰이 있는지 확인

## 주간 (매주 1회)

- [ ] 주간 회고 시점인지 확인 (일요일/월요일)
- [ ] MEMORY.md에 이번 주 교훈 반영했는지 확인
- [ ] memory/ 일일 로그에서 반복 패턴 → MEMORY.md로 승격할 것 있는지 확인

## 월간 (매월 1회)

- [ ] USER.md에 새로 파악된 사용자 특성 반영
- [ ] IDENTITY.md 안티패턴 목록 점검
- [ ] SOUL.md 업데이트 필요 여부 검토
