# NVC 자기대화 성찰

> Marshall Rosenberg의 비폭력대화(NVC) 4단계를 자기 자신과의 대화에 적용한 성찰 프레임워크.
> 자기 비난을 자기 이해로 전환한다.

## 적합한 상황

- **자기 비난/자책**이 나올 때 ("나는 게을러", "또 못했어", "이러면 안 되는데")
- 감정이 있는데 원인을 모를 때 ("짜증나는데 왜인지 모르겠어")
- 내면의 강요 패턴이 반복될 때 ("해야 하는데 못했어")
- 동기가 떨어질 때 — 욕구를 찾으면 동기가 자연스럽게 따라온다
- **Gibbs나 What-So What-Now What에서 자기 비난이 섞여 나올 때** 전환용으로

## 구조

| 단계 | 질문 | 초점 |
|------|------|------|
| 관찰 (Observation) | 실제로 무슨 일이 있었나? | 평가 없이 사실만 |
| 느낌 (Feeling) | 그때 어떤 감정을 느꼈나? | 감정 인식 (욕구의 신호) |
| 욕구 (Need) | 어떤 욕구가 충족/미충족됐나? | 감정 뒤의 진짜 원인 |
| 부탁 (Request) | 나에게 어떤 부탁을 할 수 있을까? | 강요 아닌 선택 |

### 관찰 (Observation)
- 카메라가 찍을 수 있는 사실만
- "게을렀다" ❌ → "오후에 2시간 누워있었다" ✅
- "못했다" ❌ → "계획 3개 중 1개 완료" ✅

### 느낌 (Feeling)
- 감정은 욕구의 신호 — 감정 자체에 좋고 나쁨 없음
- "짜증난다" → 끝이 아니라 시작. "이 감정이 뭘 가리키지?"
- 욕구 충족 시: 기쁨, 감사, 편안, 뿌듯함
- 욕구 미충족 시: 불안, 답답, 외로움, 초조

### 욕구 (Need)
- 모든 감정은 욕구에서 출발
- 주요 욕구 범주: 자율(선택, 독립), 연결(인정, 소속), 의미(성장, 기여), 안전(안정, 질서), 신체(휴식, 영양), 즐거움(놀이, 아름다움)
- 감정 → 욕구 연결이 되면 자기 비난 없이 다음을 찾을 수 있다

### 부탁 (Request)
- 강요("해야 해")가 아닌 부탁("해볼까?")
- 안 했을 때 자책이 따르면 강요, 괜찮으면 부탁
- 욕구를 충족하기 위한 구체적이고 실행 가능한 다음 행동

## 저널 기록 형식

```markdown
**[항목명]**

- `👁️ 관찰:` [평가 없이 사실만]
- `💭 느낌:` [어떤 감정을 느꼈나]
- `❤️ 욕구:` [어떤 욕구가 충족/미충족됐나]
- `🙏 부탁:` [나에게 하는 부탁 — 강요 아닌 선택]
```

## 유도 질문

- "구체적으로 어떤 상황이었어?" (평가 → 관찰 전환)
- "그때 어떤 느낌이었어?"
- "그 감정 뒤에 어떤 마음이 있는 것 같아?"
- "지금 뭐가 필요한 것 같아?"
- "나한테 어떤 부탁을 하고 싶어?" (강요 → 부탁 전환)

## 원칙

- **자기 비난을 자기 이해로 전환**하는 게 핵심 목적
- 관찰 단계에서 평가가 섞이면 부드럽게 분리 ("구체적으로?")
- 욕구를 찾으면 감정이 자연스럽게 완화됨
- 부탁은 항상 "No"가 가능한 형태 — 안 해도 자책 없음
- 다른 프레임워크로 성찰하다가 자기 비난이 나오면 이 프레임워크로 전환 가능
