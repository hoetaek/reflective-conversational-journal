# What-So What-Now What

> Rolfe et al. (2001)의 성찰 프레임워크. 경험을 단계적으로 분석하여 다음 행동으로 연결한다.

## 적합한 상황

- 시행착오나 실수가 있었을 때
- 외부 피드백을 받았을 때
- 기대와 결과가 달랐을 때
- **기본 프레임워크** — 다른 프레임워크가 더 적합하지 않으면 이것을 사용

## 구조

| 단계 | 질문 | 초점 |
|------|------|------|
| What? | 무슨 일이 있었나? | 사실 기술 |
| So What? | 그래서 뭘 배웠나? | 의미/교훈 발견 |
| Now What? | 다음에 뭘 다르게 할 건가? | 행동 계획 |

### What? (무슨 일)
- `📍 Context:` 상황/맥락
- `🎯 Action:` 내가 한 행동
- `→ Result:` 결과

### So What? (의미/교훈)
- `👍 좋았던 점:`
- `👎 아쉬운 점:`
- `💡 배운 점:`

### Now What? (다음 행동)
- 구체적이고 실행 가능한 다음 단계

## 저널 기록 형식

```markdown
**[항목명]**

- `🔍 What? (무슨 일):`
  - `📍 Context:` [상황]
  - `🎯 Action:` [행동]
  - `→ Result:` [결과]
- `💡 So What? (의미/교훈):`
  - `👍 좋았던 점:` [좋았던 점]
  - `👎 아쉬운 점:` [아쉬운 점]
  - `💡 배운 점:` [배운 점]
- `✨ Now What? (다음 행동):` [다음 행동]
```

## 유도 질문

- "그때 정확히 뭘 했어?"
- "기대한 건 뭐였어?"
- "결과가 달랐다면 왜 그렇게 됐을까?"
- "다음에 비슷한 상황이 오면 어떻게 할 거야?"

## 원칙

- 모든 항목을 채울 필요 없음 — 의미 있는 것만
- "별로", "잘 모르겠어" → 존중하고 What만 기록해도 됨
