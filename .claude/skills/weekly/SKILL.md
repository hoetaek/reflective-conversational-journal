# Weekly Skill

일주일간 저널에서 패턴을 발견하고 인사이트 도출.

> SOUL.md/IDENTITY.md 따르기: 반응 먼저, 질문은 보조

## 프로세스

### 1. 저널 파일 수집
- 지난 월요일~일요일 7일간 저널 읽기
- 프로필.md도 함께 읽기 (맥락 이해)
- 파일 경로: Obsidian daily-notes 설정에 따름

### 2. 패턴 분석
- 반복되는 테마, 감정, 활동 패턴
- 개별 사건들을 연결하여 더 큰 의미 발견
- 프로필.md의 가치관/목표와 연결

### 3. 초안 생성 (템플릿 항목)
- **완료한 프로젝트들**: 구체적 성과, 완성한 작업
- **좋아했던 것들**: 즐거웠거나 만족스러웠던 경험
- **잘한 것들**: 성취나 잘 처리한 상황
- **감사한 것들**: 고마웠던 사람이나 상황
- **아쉬웠던 것들**: 개선이 필요하거나 아쉬웠던 부분
- **배운 것들**: 새로운 깨달음이나 학습 내용
- **다음 주 인사이트**: 패턴 분석 결과와 개선 방향

### 4. 사용자와 함께 개선
- 초안 보여주기
- "어떤 부분을 더 자세히 써야 할까? 빠뜨린 중요한 내용이 있어?"
- 피드백 받아 함께 수정/보완

### 5. 파일 생성
- 경로: 저널 폴더 내 `Weekly/YYYY 회고 W[ISO주번호].md`
- 완성된 내용으로 파일 생성

## 대화 시작 예시

"이번 주 저널들을 읽어보니까 흥미로운 패턴들이 보이네. 한번 같이 정리해볼까?"

## 핵심

- 패턴 발견과 연결에 집중
- 사용자와 대화하며 완성
- 템플릿은 가이드일 뿐, 유연하게 조정
