# 연결 질문 패턴

대화 시작 시 상황에 맞는 연결 질문 생성:

## 1. 시간축 연결 (과거→현재→미래)

"어제 {주요사건}에서 느꼈던 걸 바탕으로, 오늘 {현재상태}을 하면서 어떤 새로운 발견이나 변화가 있었어? 그게 {목표} 목표와 어떻게 연결되는 것 같아?"

## 2. 패턴 연결

"전에도 {반복패턴} 패턴이 있었는데, 오늘 {현재상황}에서도 비슷한 게 나타났어? 아니면 뭔가 다르게 접근해봤어? 이 패턴에서 뭘 배울 수 있을까?"

## 3. 가치-선택 연결

"평소 {명시가치}을 중요하게 생각한다고 했는데, 오늘 {실제선택} 선택을 한 것을 보면 어떤 생각이 들어? 이 간극에서 뭔가 새롭게 발견한 게 있을까?"

## 대화 시작 흐름

1. (오전 9시 이전) 어제 주요 사건 간단히 언급
2. 최신화된 월간/주간 목표 상태 요약
   - 목표 간 연결성: "연결됨", "부분연결", "분리됨"
   - 진행 상태: "활발", "진행중", "미미", "진행없음"
3. 연결 질문 생성
