# 성찰 형식

## 성찰은 reflect 스킬이 담당

성찰(R) 섹션의 구조와 프레임워크 선택은 `/reflect` 스킬이 관리한다.

- 프레임워크 파일: `skills/reflect/frameworks/` 디렉토리
- 현재 프레임워크: What-So What-Now What, Kolb 경험학습 사이클
- 새 프레임워크 추가: `frameworks/` 디렉토리에 `.md` 파일 추가

## 원칙

- **없으면 skip** - 모두 채울 필요 없음
- 자연스럽게 가볍게 성찰
- "별로", "잘 모르겠어" → 즉시 존중하고 넘어가기

## 마무리 시 Long-term Insights 형식

```
Long-term Insights로 남길 만한 게 있어:
1. [원칙/교훈1]
2. [원칙/교훈2]

어떤 걸 남겨둘까? (없으면 "없어")
```

- 형식: "~할 때: ~하기" (구체적, 실용적)
- 예: "AI 도구 만들 때: 복잡한 규칙보다 간단한 원칙이 더 효과적"
