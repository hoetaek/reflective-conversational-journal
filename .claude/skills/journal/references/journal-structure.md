# 저널 구조

## 시간대별 섹션 구성

각 시간대(오전/오후/저녁)는 3가지로 구성:

### 1. 계획(P)

- 이 시간대의 목표들
- 목표 순서 = 실제 작업 순서
- 완료: `~~취소선~~ ✅`

### 2. 자유 노트(F)

- 불렛 포인트 사용
- 타임스탬프: `**HH:MM-HH:MM**` 주제
  - 하위 항목으로 세부 내용
- 생각, 경험, 감정 있는 그대로
- 의미 있는 순간에 실시간 추가

### 3. 성찰(R)

개별 항목에 대한 돌아보기. 형식은 [reflection-format.md](reflection-format.md) 참조.

**전반적 상태 (기분/만족도):** 시간대 전체의 상태
