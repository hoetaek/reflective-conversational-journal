# 저널 구조

## 섹션 구성

각 저널은 3가지 섹션으로 구성:

### 자유 노트

- **서술형 문단**으로 작성 (불릿+타임스탬프 로그 아님)
- 주어 = "나" (작업이 아니라 내가 뭘 했는지)
- 감각/반응/감정 포함 ("시원했다", "기분 좋았다")
- 3개월 뒤 다시 읽었을 때 그날이 떠오르는 수준
- 소프트 프롬프트 질문이 가이드로 있음

### 성찰

개별 항목에 대한 돌아보기. 형식은 [reflection-format.md](reflection-format.md) 참조.

### 장기 인사이트

오늘의 경험에서 발견한 핵심 원칙/교훈. 있을 때만 작성.
