# AI 장기 메모리

> **Curated 장기 지식** - 선별된 요약, 결정, 교훈만 기록
> 상세 정보는 `memory/YYYY-MM-DD.md`에 보관

## 메모리 구조

| 계층 | 파일 | 성격 |
|------|------|------|
| **장기 기억** | `MEMORY.md` (이 파일) | Curated — 결정, 교훈, 패턴만 |
| **일일 로그** | `memory/YYYY-MM-DD.md` | Raw — 세션 기록, 대화 요약, 작업 내용 |

**원칙:**
- 파일이 진실의 원천이다 — 모델은 디스크에 쓴 것만 "기억"한다
- 컨텍스트 압축 전에 중요한 정보를 파일로 저장한다 (Memory Flush)
- 일일 로그에서 반복되는 패턴 → 이 파일로 승격
